## How to Run

### 1. Install Dependencies
Make sure you have the following libraries installed:

```bash
pip install fastapi google-genai python-dotenv markdown uvicorn
```

### 2. Run
```bash
   uvicorn main:app --reload
```
### 3.Open
```bash
  http://127.0.0.1:8000/
```
